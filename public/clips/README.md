Place your cricket video clips here, organized by player.

Example structure:
  clips/
    virat-kohli/
      clip1.mp4
      clip2.mp4
    ms-dhoni/
      clip1.mp4
    sachin-tendulkar/
      clip1.mp4

IMPORTANT:
- Use short clips (3-10 seconds each)
- Vertical (9:16) clips work best for Shorts
- At least 1 clip per player referenced in daily.json
- File must be named clip1.mp4, clip2.mp4, etc.
- The folder name must match the "player" field in daily.json
